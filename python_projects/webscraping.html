Using Python for webscraping

This is a Python program used to scrape an image from a site that does not dynamically load content (infinite scrolling).


  • Uses Python requests and BeautifulSoup
  1. User inputs the desired Github user's username
  2. Requests sends a request to the Github webpage
  3. BeautifulSoup locates and selects the user's profile image
  4. Prints the url for the Github user's profile picture
