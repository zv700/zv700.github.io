Using Python to run a binary search

Python program that searches through a numbered, ordered(least to greatest) list using a "divide and conquer" method.

  1. Program searches for a number starting with the midpoint of the list
  2. If the midpoint number fits the search criteria, the search ends.
  3. If the midpoint does not fit criteria, the list is split in half between greater than or less than the midpoint.
  4. Program only searches through the half where the number could be, once again starting from the midpoint.
  5. Repeat process until target fits criteria
