Self project: FFMPEG video cut GUI project

Problem: using video editing software to cut video clips can be time consuming

  • Cutting video (i.e. saving a small portion of a large video file as its own video clip) using video editing software can take anywhere from 5 minutes to an hour depending on the length of the video clip you are trying to make and how long the original video is.
  • Video editing software by default reencodes video before you can save the clip as a video file outside of your project file.
  • Reencoding video is unecessarily time consuming if you aren't making any meaningful edits to the clip being produced.

Solution: Use FFMPEG to cut video without reencoding.

  • FFMPEG can cut video without reencoding which usually takes *seconds.*

Subproblem: FFMPEG must be run from commandline

Using FFMPEG on Windows PC
  1. open commandline
  2. change directory to ffmpeg.exe's location [cd users/username/videos/ffmpeg.exe]
  3. open up my FFMPEG command notes .txt file
  4. copy + paste the name of the video file I want to cut into the prewritten FFMPEG command template code, twice. Once for the input location and once for the output location.
  5. change the ouput location's file name so it doesn't overwrite the original file. [video.mp4 >>> video-cut.mp4]
  6. play the video file to find times for the beginning and end of the portion I want to cut out.
  7. type in beginning and end times in the template code
  8. copy + paste ffmpeg command template code into commandline. Press Enter to run. Video outputs to same directory as FFMPEG.exe

Subproblem's solution: make a GUI for cutting video via FFMPEG

  • It would save a little more time to be able to just be able to open a GUI to run FFMPEG for this specific purpose since I do not use any of FFMPEG's other features.
  • With a GUI, I could skip the steps where I open commandline and open the text file. Maybe two or three less instances of copy + paste-ing, which could save a minute or two.

Program outline:

  • use Python Tkinter
  • 4 text input boxes:
    1. original video's file name
    2. start time of cut
    3. end time of cut
    4. ouput video clip's new file name
  • button that runs program
  • Store the text input into the input boxes as 4 string variables
  • Have the FFMPEG command template written with fstrings that will accept each of the 4 variables
  • Program runs FFMPEG from commandline and inputs the completed template command into FFMPEG to have it cut the video
