Python famous block puzzle game clone

Game instructions

Game controls

  • utilizes WASD controls
  • press A and D to move blocks left and right
  • press W to rotate the block
  • press D to instantly drop the block
  • press ESC to close game window

Game rules

  • arrange blocks in full horizontal lines to clear/erase the row of blocks
  • clearing rows increases score
  • game ends when a block touches the top of the playing field
